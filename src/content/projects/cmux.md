## Overview

**cmux** is a terminal interface designed around AI coding agents.

It uses Ghostty as a foundation and adds an agent-oriented layer: vertical tabs, notification rings, unread states, and a workflow shaped around multiple coding sessions asking for attention at different times.

## Why It Belongs Here

cmux fits the Agent shelf because it treats agent work as an interface problem. Once coding agents can run in parallel, the main challenge becomes orchestration: seeing which session matters, knowing where attention is needed, and avoiding a mess of terminal panes.
