## Overview

**Genres in Genres** studies style evolution inside a music collection.

The project uses MuQ-MuLan embeddings to map songs into a semantic music space, then clusters and visualizes how an artist's sound changes across albums and years.

## Project Shape

The system combines library scanning, embedding extraction, K-Means clustering, dimensionality reduction, radar charts, and temporal style views.

It is a course-project artifact, but it also belongs to a larger Audiofool thread: using machine listening to make musical taste, history, and structure visible.
